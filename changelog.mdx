## Upgrade Guide

Follow these steps to upgrade Orange Documentation.

<Steps>

<Step title="Backup" icon="database">

Export your current documentation site.

```bash
npx orange-docs export ./backup
```

</Step>

<Step title="Update Package" icon="download">

Install the latest version.

<CodeGroup tabs="npm,yarn">

```bash
npm install orange-docs@latest
```

```bash
yarn add orange-docs@latest
```

</CodeGroup>

</Step>

<Step title="Rebuild" icon="play">

Rebuild and deploy your site.

```bash
npx orange-docs build
npx orange-docs deploy
```

</Step>

</Steps>

## Known Issues

<Callout kind="alert">

- Mermaid diagrams may fail with complex syntax containing `{` or `}`. Use simple nodes like `A[Start] --> B[End]`.
- Custom icons outside Lucide library show as placeholders.

Report issues on GitHub.

</Callout>
