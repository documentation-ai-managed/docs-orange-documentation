## Overview

Orange Documentation provides a centralized hub where you create, organize, and publish project documentation. You streamline workflows by combining markdown editing, version control, and search capabilities into one intuitive platform. Designed for developers, product managers, and teams, it supports collaborative editing and custom branding.

<Callout kind="info">
Orange Documentation uses your brand color `#3B82F6` for consistent theming across all pages.
</Callout>

## Quick Start

Set up your first documentation space in minutes.

<Steps>
  <Step title="Create Space" icon="plus">
    Log in and select "New Documentation Space". Enter your project name and brand color `#3B82F6`.
  </Step>
  <Step title="Add Pages" icon="file-plus">
    Click "New Page" and start writing in markdown. Use frontmatter for metadata.
  </Step>
  <Step title="Publish" icon="globe">
    Preview changes, then publish to make your docs live for your team.
  </Step>
</Steps>

## Brand Integration

Integrate your `#3B82F6` brand color seamlessly.

Customize CSS variables in your space settings or via configuration files.

<CodeGroup tabs="CSS,JSON">
  ```css
:root {
  --brand-primary: #3B82F6;
  --brand-secondary: #1D4ED8;
}

.header {
  background-color: var(--brand-primary);
}
  ```
  ```json
{
  "theme": {
    "primaryColor": "#3B82F6",
    "logo": "/assets/orange-logo.svg"
  }
}
  ```
</CodeGroup>

<Expandable title="Advanced Theming" default-open="false">
For deeper customization, override MDX components with your React setup.

```tsx
// theme.tsx
import { ThemeProvider } from '@orange/docs';

const customTheme = {
  colors: { primary: '#3B82F6' }
};

<ThemeProvider theme={customTheme}>
  {/* Your docs */}
</ThemeProvider>
```
</Expandable>
